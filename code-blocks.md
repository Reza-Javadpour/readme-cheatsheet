# Readme Code Block

With these keys, you can make a pretty syntax highlighter for readme code blocks:

## How to Using:
#### Usage:
```
    ```javascript
        console.log('Reza Javadpour')
    ```
```
#### Output
```javascript
  console.log('Reza Javadpour')
```
